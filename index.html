HTML as One of My Core Skills

HTML (HyperText Markup Language) is the standard language for creating web pages. It uses elements and tags to organize and structure content.

Structural Elements

  • <header> – page or section header
  • <main> – wraps main content
  • <section> – groups related content
  • <article> – self-contained content
  • <footer> – page or section footer

Text Formatting Elements

  • <h1>–<h6> – headings
  • <p> – paragraphs
  • <strong> – important text
  • <em> – emphasized text

Lists, Links, & Images

  • <ul> / <ol> / <li> – lists
  • <a href="URL"> – hyperlink
  • <img src="file.jpg" alt="description"> – image

Tables & Forms

  • <table>, <tr>, <td>, <th> – tables
  • <form>, <input>, <label>, <textarea>, <button> – forms
