HTML as One of My Core Skills

HTML (HyperText Markup Language) is the backbone of web development. It uses elements and tags to structure content meaningfully and semantically.

Structural Elements

  • <header> – for page or section headers
  • <main> – wraps main content
  • <section> – groups related content
  • <article> – self-contained content blocks
  • <footer> – page footer

Text Formatting Elements

  • <h1>–<h6> – headings
  • <p> – paragraphs
  • <strong> – important text
  • <em> – emphasized text

Lists, Links, & Images

  • <ul> / <ol> / <li> – lists
  • <a href="URL"> – hyperlinks
  • <img src="file.jpg" alt="description"> – images

Tables & Forms

  • <table>, <tr>, <td>, <th> – tables
  • <form>, <input>, <label>, <textarea>, <button> – forms
